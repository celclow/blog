Windows 10にpingが通らない

  • 2015-08-24
  • コンピュータ  
  • Windows  
  • ボケ防止  

aterm ルータに Windows 10 と OS X Yosemite がぶら下がってる環境で、Windows 向けの ping が通らない。

Windows 側で、以下の 2 点を変更することで解決。

  • プライベートネットワークに変更
    • デフォルトがパブリックネットワークになってる？(未確認)
    • スタート> 設定 > ネットワークとインターネット > イーサネット1 > アイコンをクリック > デバイスとコンテンツの検索のトグル
    • オンならプライベートネットワーク、オフならパブリックネットワーク
  • ファイアウォールの設定
    • コントロールパネル > Windows ファイアウォール > 詳細設定
    • 受信の規制
    • ファイルとプリンターの共有 (エコー要求 – ICMPv42 受信) プロファイル: プライベート, パブリック

みたいな感じにすると、通るように。

  1. 有線接続の場合。 [return]
  2. IPv6 なひとは ICMPv6 かもしれない。 [return]
